Overview
========

This is a django project that provides SOS support for Grafana. It relies on sosdb-ui
SOS, and apache.

Installation
============
Install Dependencies:
    sosdb-ui
    SOS
    sosds (Grafana datasource for SOS)
    Grafana

Annotations
============

Query format is as such:
	store&ptn_id&job_id&comp_id
